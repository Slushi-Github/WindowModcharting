# Supported FNF' Engines

## Psych Engine

Define `WM_ENGINE` as `PYSCH` in `Project.xml`, and define `WM_ENGINE_VERSION` as `0.7` if you are using Psych 0.7 or higher, or `0.6` if you are using Psych 0.6 or lower.

## Codename Engine

Define `WM_ENGINE` as `CODENAME` in `Project.xml`, `WM_ENGINE_VERSION` is not required.

## ALE Psych

Define `WM_ENGINE` as `ALE_PSYCH` in `Project.xml`, `WM_ENGINE_VERSION` is not required.

## Base Game / V-Slice

Define `WM_ENGINE` as `V_SLICE` in `Project.xml`, `WM_ENGINE_VERSION` is not required.
